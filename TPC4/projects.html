Intelligent Systems

Python
Done

This project focuses on the design and implementation of a multi-agent system (MAS) to monitor patients in Intensive Care Units (ICUs). The system leverages intelligent agents to interpret patients' vital signs, detect critical changes, and coordinate rapid responses to ensure timely and specialized medical care. The SPADE library is used to implement the agents in a distributed architecture, enabling real-time monitoring and efficient communication between agents. The system operates in a hospital setting, where critically ill patients are continuously monitored, and aims to improve patient outcomes by ensuring immediate and appropriate medical responses.
